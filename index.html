// features

What AstroCore can do

A tool for CNC operators and hobbyists — from simple engravers to more complex 3-axis machines. Connects via USB/Serial or TCP/IP.

3D Visualizer
Real-time preview of G-code toolpaths. Rotate, zoom, and isometric view. Vertex and FPS counter.
Heightmap
Automatic surface probing with bicubic interpolation. 2D color grid and 3D contour map views.
Jog & Joystick
Move X/Y/Z axes step-by-step or continuously — using keyboard, gamepad or joystick. Controller support is the key new feature over Candle.
G-code viewer
Command list with color-coded status (in queue / sending / done). Auto-scroll and comment filtering.
USB / Serial & TCP/IP
Connect via a standard USB/serial port or over TCP/IP — useful for networked controllers like ESP32-based boards.
Virtual machine modes
Simulate uCNC, grblHAL and FluidNC without any hardware connected. Test programs safely on your computer.
Speed override
Feed rate, rapid speed and spindle speed sliders — adjust on the fly while a program is running.
GRBL console
Direct access to the device console. See controller responses and send commands manually.
Camera
Built-in camera view inside the interface. Watch the machining process without switching windows.
Customizable interface
Dark and light theme. Panels can be moved and customized. Interface scales to different screen sizes.
Log browser
A dedicated view for controller communication history. Diagnose issues and track program execution.
G-code editing
Built-in G-code file editor. Load, edit and save files without switching to external tools.

Built on solid foundations,
rebuilt from the ground up

AstroCore grew out of the Candle experimental branch. The goal: bring the codebase into the modern era — cleaner architecture, new features, active maintenance, and fixes for long-standing issues that the original project never addressed.


Supported firmware: GRBL 1.1, uCNC, grblHAL, FluidNC. Feedback, bug reports and pull requests are welcome.
